# User Manual - Lightweight PM System

## Access
Open index.html in a browser. Ensure the Firebase config in app.js is set to your project.

## Create Task
- Enter title, description, priority, due date, then click 'Add Task'.

## View Tasks
- Tasks will appear in the Tasks section.

## Toggle Status / Delete
- Use 'Toggle Status' to mark Done/To Do. Use 'Delete' to remove a task.

## Notes
- This is a prototype. Data persists in Firestore.
